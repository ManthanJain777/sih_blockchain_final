2. Upload Page (`#upload`)
File: `/pages/UploadPage.tsx`

Components:
`WalletConnection` - MetaMask wallet connection
`FileUploader` - File selection and hash generation
`MetadataDisplay` - JSON metadata preview
Real-time clock display
Transaction details display (after blockchain submission)

Key Functions:
`handleConnectionChange(connected, address)` - Wallet connection state
`handleFileData(data)` - File data management
`handleUploadToIPFS()` - IPFS upload simulation
`handleSubmitToBlockchain()` - Blockchain submission simulation

Data Flow:
User connects wallet
User selects file
Generate SHA256 hash + IPFS CID + Geo-tags
Display metadata aggregation (JSON)
Upload to IPFS
Submit to blockchain
Show transaction details
